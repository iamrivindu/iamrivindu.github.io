What I'm upto these days..

Well, as always, I'm a keen learner. I like to learn new things. These days I'm working on my Semester 05 Project which is essentially a Statistical Summarization application for developers. We have to fetch from certain web services like Github, Stackoverflow and Twitter, and display those information about a certain developer in such a way that a HR division of an organization would be able to make decisions on that particular developer. This, in my opinion, would be heavily useful for organizations which are overlooking on the interns which they recruit. This can be a procedure which the HR division can carry on before calling the interns for interviews. You can read about this project from my blogpost here.
